# cb-log
A very easy ```console.log()```

## Instalation
```
npm i cb-log
```
## Import
```javascript
var cb_log = require("cb-log");
```
## Usage
```javascript
cb_log.log();
```
### Displays plain text, like a console.log().
```javascript
cb_log.logln();
```
### Like ```cb_log.log()```, but starts with ```\n```
```javascript
cb_log.logSuccess():
```
### Displays a lightgreen text
```javascript
cb_log.logInfo();
```
### like ```cb_log.logSuccess()```, but lightcyan color
